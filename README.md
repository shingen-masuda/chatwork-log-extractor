# Chatwork Log Extractor

Chatworkの特定のルームのやり取りを、返信関係や引用、宛先（To/CC）を保持したままJSON形式で抽出するブラウザ用スクリプトです。

## 特徴

- **高精度な解析**: Reactベースの最新Chatwork UI（Biwa）に対応。
- **返信関係の保持**: どのメッセージへの返信かを `replyTo` フィールドに記録。
- **引用の再現**: `[qt]` タグや引用ボックスを `> ` 記号で整形。
- **宛先の分離**: ToやCCで指定されたメンバーを個別のフィールドに抽出。
- **日付の自動補完**: 表示が省略されているメッセージにも正しい日付を付与。
- **クリーンな抽出**: アイコンの代替テキストなどの不要なノイズを除去。

## 使い方

1. ブラウザで対象のChatworkルームを開きます。
2. `F12` キー（または右クリック > 検査）でデベロッパーツールを開き、「Console」タブを選択します。
3. 初めて実行する場合、`allow pasting` と入力して Enter を押し、貼り付け制限を解除します。
4. 本リポジトリの `extract_chatwork_logs.js` の中身をすべてコピーしてコンソールに貼り付け、Enter を押します。
5. 抽出が開始されます。自動で過去に遡り、2年前までのログを収集します。
6. 完了すると、`ルーム名_実行日_期間.json` という名前でファイルがダウンロードされます。

## 出力フォーマット (JSON)

```json
{
  "id": "メッセージID",
  "time": "2024年10月21日 15:33",
  "user": "投稿者名",
  "to": "宛先メンバー名",
  "cc": "CCメンバー名",
  "replyTo": "返信先メッセージID",
  "text": "メッセージ本文（引用は > 付き）"
}
```

## 注意事項

- 本スクリプトは読み取り専用であり、メッセージの編集や送信は行いません。
- 大量のメッセージがある場合、実行に時間がかかることがあります（1スクロールごとに2秒の待機時間を設けています）。
- 実行中にブラウザを閉じたりリロードしたりすると、抽出が中断されます。
